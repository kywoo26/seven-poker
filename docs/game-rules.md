# 한국식 세븐 포커 규칙

## 게임 개요

- 플레이어: 2-5명
- 카드: 52장 (조커 없음)
- 목표: 7장 중 5장으로 최고 족보 완성

## 카드 배분

### 초기 배분 (4장 → 3장)
1. 각 플레이어에게 4장 배분 (모두 비공개)
2. 각 플레이어가 1장을 버림 (discard)
3. 남은 3장 중 1장을 선택하여 오픈
4. 결과: 2장 히든 + 1장 오픈

### 4th~6th 스트리트
- 오픈 카드 기준 가장 높은 패 소유자부터 카드 수령
- 받은 카드는 오픈 상태

### 7th 스트리트 (리버)
- 마지막 카드는 히든으로 배분
- 최종: 3장 히든 + 4장 오픈

## 베팅

### 베팅 라운드
1. **1차**: 초기 배분 후 (오픈 카드 선택 후)
2. **2차**: 4th 카드 배분 후
3. **3차**: 5th 카드 배분 후
4. **4차**: 6th 카드 배분 후
5. **5차**: 7th 카드 배분 후

### 베팅 순서
- 오픈 카드 기준 가장 높은 패 소유자가 선
- 동점 시 무늬 순서: 스페이드 > 다이아 > 하트 > 클로버

### 베팅 액션
| 액션 | 설명 |
|------|------|
| 체크 | 베팅 없이 넘기기 (첫 베터만 가능) |
| 콜 | 앞사람과 동일 금액 베팅 |
| 하프 | 현재 팟의 50% 베팅 |
| 풀/올인 | 현재 팟의 100% 베팅 |
| 다이/폴드 | 게임 포기 |

## 족보 (높은 순)

### 1. 로열 스트레이트 플러시
같은 무늬 A-K-Q-J-10

### 2. 백 스트레이트 플러시
같은 무늬 A-2-3-4-5

### 3. 스트레이트 플러시
같은 무늬 연속 5장 (예: ♠6-7-8-9-10)

### 4. 포카드
같은 숫자 4장 (예: K-K-K-K-X)

### 5. 풀하우스
트리플 + 페어 (예: Q-Q-Q-7-7)

### 6. 플러시
같은 무늬 5장 (연속 아님)

### 7. 마운틴
A-K-Q-J-10 (무늬 상관없음)

### 8. 백스트레이트
A-2-3-4-5 (무늬 상관없음)

### 9. 스트레이트
연속 5장 (무늬 상관없음)

### 10. 트리플
같은 숫자 3장

### 11. 투페어
페어 2개

### 12. 원페어
같은 숫자 2장

### 13. 하이카드 (노페어)
아무 족보 없음, 가장 높은 카드로 비교

## 승패 결정

1. 족보 비교
2. 동일 족보 시 숫자 비교 (A가 가장 높음)
3. 동일 숫자 시 무늬 비교

## 무늬 순서

스페이드(♠) > 다이아몬드(♦) > 하트(♥) > 클로버(♣)

## 특수 규칙

### 스트레이트에서 A의 역할
- A-K-Q-J-10: 마운틴 (A가 높음)
- A-2-3-4-5: 백스트레이트 (A가 낮음)
- K-A-2-3-4: 불가 (스트레이트 아님)

### 7장 중 5장 선택
최고의 5장 조합을 자동 선택 (플레이어가 직접 선택 안 함)
